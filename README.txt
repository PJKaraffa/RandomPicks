RANDOM PULL SYSTEM

FILES
- index.html
- style.css
- script.js

HOW TO USE
1. Keep all three files in the same folder.
2. Open index.html in a browser.
3. Upload a CSV file.
4. Select the column that uniquely identifies each record.
5. Select the main name/display column.
6. Enter the number of records to pull.
7. Click Run Random Pull.

FEATURES
- CSV upload and preview
- Random selection without duplicate records in the same pull
- Automatic exclusion of selected records from future pulls
- Manual exclusion of checked records
- Restore one excluded record or restore all
- Search eligible and excluded records
- Current selection export
- Complete pull-history export
- Browser storage so data and history survive refreshes
- Full reset option

IMPORTANT
The system saves data in the browser using localStorage. It does not use a database.
For the strongest duplicate prevention, choose a column with a unique value for every person,
such as Employee ID, Student ID, SASID, email address, or another permanent identifier.
